# **App Name**: Modern Weebly Clone

## Core Features:

- Page Creation: Allow users to create and manage multiple pages.
- Drag and Drop Interface: Enable drag-and-drop functionality for adding and arranging elements on each page.
- Element Library: Provide a library of customizable elements (text, images, buttons, etc.) to add to pages.
- Responsive Design: Ensure the website is responsive and looks good on all devices (desktop, tablet, mobile).
- Theme Selection: Offer a selection of modern, clean themes to choose from.
- Customization: Allow users to modify existing CSS and JS to meet custom design specs.

## Style Guidelines:

- Primary color: Muted blue (#64B5F6) to convey trustworthiness and calm.
- Background color: Very light blue (#E3F2FD) for a clean, uncluttered look.
- Accent color: Soft purple (#9575CD) to highlight interactive elements without being jarring.
- Body and headline font: 'Inter', sans-serif, for a modern and readable look.
- Use a set of minimalist, line-based icons to represent common actions and elements.
- Implement a grid-based layout system for consistent spacing and alignment.
- Incorporate subtle animations for transitions and interactive elements to enhance the user experience.
